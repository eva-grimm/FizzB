                        // Gets the numbers from the start/end value input fields
                        function getValues() {
                            // get the values from the form and attempt to convert string to number
                            let startNumber = Number(document.getElementById('start-value').value);
                            let endNumber = Number(document.getElementById('end-value').value);

                            // validation
                            if (isNaN(startNumber) == true || isNaN(endNumber) == true ) { 
                                // if we don't have numbers
                                Swal.fire({
                                    title: 'Oops!',
                                    text: 'Sorry, but Fizz, B only works with real numbers',
                                    icon: 'error',
                                    backdrop: false
                                });
                            } else if (startNumber > endNumber) {
                                // if we have a starting number greater than the ending number 
                                Swal.fire({
                                    title: 'Oops!',
                                    text: 'The starting number must be less than the ending number',
                                    icon: 'error',
                                    backdrop: false
                                });
                            } else if (endNumber > 100) {
                                // if we have a ending number greater than 100
                                Swal.fire({
                                    title: 'Oops!',
                                    text: 'Sorry, but no ending number greater than 100',
                                    icon: 'error',
                                    backdrop: false
                                });
                            } else {
                                // create an array of the numbers in range
                                let range = generateNumbers(startNumber, endNumber);

                                // display the numbers in range
                                displayNumbers(range);
                            }
                        }

                        // Creates every number in the chosen input range
                        function generateNumbers(start, end) {
                            let range = [];

                            // create an array including only the numbers in range
                            for (let number = start; number <= end; number++) {
                                range.push(number);
                            }

                            return range;
                        }

                        // Puts the numbers on the page
                        function displayNumbers(numbersToDisplay) {
                            let tableHtml = '';
                            let tableRowHtml = '';

                            for (let i = 0; i < numbersToDisplay.length; i++) {
                                let currentNumber = numbersToDisplay[i];

                                // if the current number is one greater than a number divisible by 5, 
                                // or is otherwise the number 1, then begin a new row
                                if (currentNumber % 5 == 1) {
                                    tableRowHtml = `<tr>`;
                                }

                                // if the current number is divisible by 2, then make it bold
                                if (currentNumber % 2 == 0) {
                                    tableRowHtml = `<td><strong>${currentNumber}<strong></td>`
                                } else {
                                    tableRowHtml = `<td>${currentNumber}</td>`
                                }

                                // if the current number is disivible by 5, then end the row
                                if (currentNumber % 5 == 0) {
                                    tableRowHtml += `</tr>`;
                                }

                                tableHtml += tableRowHtml;
                            }

                            document.getElementById('results').innerHTML = tableHtml;
                        }

The code is structured in three functions

getValues

This function gets the values passed from the website form and attempts to convert them to numbers. From there, it validates that the values are valid for running the rest of the function. There are four checks in total:

  • Were numbers passed? Although the form is set to only allow numbers, some browsers may not respect that limitation, resulting in the user providing values that aren't numbers.
  • Is the starting number greater than the ending number? This applet is designed to provide the numbers in a range, and that means our starting number can't be greater than the ending number.
  • Is the starting number greater than 0? Is the ending number greater than 100? Although most browsers could handle far greater ranges than 100 numbers, this would lead to a results list that is unwieldy. To avoid this, the starting and ending numbers are capped at 1 and 100 respectively.

Once these checks have been performed, the function then calls generateNumbers(), passing it the start and end values and receiving back an array. The function then passes that array on to displayNumbers().

generateNumbers

This function is the simplest but is no less integral. It receives a start number and end number as arguments and creates a blank array.

The function then uses a for-loop to fill that array with numbers. This for loop sets its index to the starting number and will run iteratively until the index is greater than the ending number. While running, the for-loop pushes the current index into the array, which will ultimately result in an array filled with numbers beginning with the start number and concluding with the end number.

Once the array of numbers has been generated, it returns this array to getValues().

displayNumbers

The final function of the applet, this function's purpose is to render the array into html to be displayed on the applet page. It receives an array as an argument and begins by initializing two variables with blank strings. These variables are a main string of HTML that will be passed to the applet page, and a current string of HTML that will be built up over the course of a for-loop depending on what the current number is and ultimately appended to the main HTML string at the end of the loop.

Once the function has initialized its variables, it begins a for-loop. The loop begins with an index of 0 and works iteratively until the index is equal to the length of the array. The loop begins by initializing a variable as the current number, which is set to the current index of the array. It then begins constructing the html to be passed to the website in four steps:

  • Beginning a row: The function checks whether the modulus/remainder of the current is equal to 1. What this means practically is the function is checking if the current number is one greater than a number divisible by 5 or otherwise the number one. If that is the case, then it appends the HTML code to the string
